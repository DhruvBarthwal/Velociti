# Velociti – AI Website Template Generator

Velociti is an AI-powered website template generator that creates complete website code and a live preview from a single natural language prompt.

Instead of manually setting up layouts, components, and styling, users can describe what they want — and Velociti instantly generates a ready-to-use website template.

## Features

- Single-prompt website generation

- Generates complete frontend code (HTML/CSS/JS or framework-based)

- Live preview of the generated website

- Clean, modern UI templates

- Fast iteration — regenerate and tweak designs quickly

- Developer-friendly output (editable code)
